# 📚 StudyGrind - Innovative Learning Platform

StudyGrind is a comprehensive learning management system that connects students, teachers, and administrators in one seamless platform. It provides tools for assignment management, quiz creation, timetable organization, and resource sharing.

## ✨ Features

### For Students
- 📅 **Personal Timetable** - Create and manage your own class schedule
- 📝 **Assignment Submission** - Submit assignments online with text or files
- 📊 **MCQ Quizzes** - Take auto-graded multiple-choice quizzes
- 📚 **Study Notes** - Access learning resources from teachers
- 📈 **Performance Tracking** - View your quiz scores and pending assignments

### For Teachers
- ✏️ **Assignment Creation** - Create and publish assignments with due dates
- 🎯 **Quiz Builder** - Create custom MCQ quizzes with automatic grading
- 📋 **Grading System** - Grade student submissions with feedback
- 📎 **Resource Upload** - Share notes and learning materials
- 👥 **Student Management** - View student submissions and performance

### For Administrators
- 👨‍🏫 **Teacher Management** - Add and manage teacher accounts
- 👩‍🎓 **Student Directory** - View all registered students
- 📊 **Platform Analytics** - Monitor platform activity and user engagement
- ⚙️ **System Settings** - Configure platform settings

## 🚀 Technology Stack

- **Backend**: Flask (Python)
- **Database**: SQLite (Development) / PostgreSQL (Production)
- **Authentication**: Flask-Login with Google OAuth support
- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: Font Awesome 6
- **Charts**: Chart.js
- **Deployment**: Vercel / Render / PythonAnywhere

## 📋 Prerequisites

- Python 3.11 or higher
- pip (Python package manager)
- Git (optional, for version control)
