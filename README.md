StudyGrind - Learning Management System
A comprehensive web application designed to facilitate seamless interaction between teachers and students for educational content management.

📋 Overview
StudyGrind is a learning management system that enables teachers to share educational materials and assignments while providing students with a centralized platform to access resources and track their academic progress.

✨ Features
👨‍🏫 Teacher Features
Notes Management: Create, edit, and organize teaching notes with public/private visibility options

Resource Sharing: Upload textbooks, study materials, and educational resources in multiple formats

Link Management: Share useful external links and resources with students

Assignment Creation: Create and manage assignments with due dates and grading rubrics

Student Overview: View and manage enrolled students

Calendar Management: Schedule classes, exams, and important deadlines

Grading System: Review and grade student submissions with feedback

👨‍🎓 Student Features
Personal Notes: Create and manage private study notes

Resource Access: View and download teacher-shared materials

Assignment Portal: View pending assignments and submit work

Progress Tracking: Track grades and feedback on submissions

Calendar View: Access all academic events and deadlines

Resource Library: Browse available textbooks and study materials

Link Repository: Access teacher-curated educational links
